# Para executar a aplicação basta:
1 - Clonar o repositório;
2 - Abrir a pasta conténdo os arquivos principais;
3 - Executar o Prompt de comandos na pasta;
4 - Executar o comando code .;
5 - O Visual Code será exibido, basta abrir o terminal, selecionar o gitBash ou powerShell e executar o comando ng serve;
6 - Copiar o localhost + porta e colar na barra de pesquisa do navegador.
